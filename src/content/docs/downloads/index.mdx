## Build From Source

```bash
git clone https://github.com/connor12858-site/P2PNetwork.git
cd P2PNetwork/GoP2P
go build ./cmd/relay
go build ./cmd/user
```

This generates local relay and user binaries using your current Go toolchain.

## Release Publishing Notes

- Publish tagged releases in GitHub to populate this page automatically.
- Attach platform-specific binaries to release assets when available.
