## What the versions tell us

From the shipped binaries and the current code:

- v1.0 corresponds to the bootstrap and peer CLI pair
- v1.1 adds the desktop GUI and the newer peer/bootstrap binaries
- the published binaries are currently Windows executables

That is the safest release summary available from the repo itself today.

## Suggested changelog format for future releases

When the repo starts tagging releases with notes, this page should list:

- version
- release date
- changed binaries
- protocol or compatibility changes
- user-facing fixes
